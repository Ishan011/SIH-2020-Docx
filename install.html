Web Technology

Web technology refers to the means by which computers communicate with each other using markup languages and multimedia packages.

  • ExpressJS is used to connect to the server and then renders the elements.
  • Javascipt for backend connection between expressjs file and server side files
  • jQuery is a JavaScript library designed to simplify HTML DOM tree traversal and manipulation, as well as event handling, CSS animation, and Ajax.
  • React components implement a render() method that takes input data and returns what to display.

Install node js [1]
